Cab Booking System

Type: Academic Project

The Cab Booking System is a full-stack web application that allows users to book rides efficiently and conveniently. It includes features like driver and user registration, fare estimation, real-time route updates, and admin-level trip history management. Designed for seamless operation with intuitive UI and robust backend services.

Roles & Responsibilities:

  • Database Design: Created MySQL schemas for drivers, users, trip logs, and fare history.
  • Routing Logic: Implemented real-time distance calculation using JavaScript APIs and Flask backend.
  • Frontend Integration: Developed dynamic and responsive UI with HTML, CSS, and JS for live updates.
  • Backend Development: Built RESTful APIs using Python Flask and integrated MySQL with secure endpoints.
  • Deployment: Connected frontend to server, deployed on Vercel and configured remote DB access using Clever Cloud.

Tools & Technologies:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python, Flask
  • Database: MySQL
  • Hosting: Vercel (Frontend), Clever Cloud (Database)
